# Stack

HTML, CSS, JavaScript ES6, React.js, Firebase

### 커밋컨벤션

| 태그이름   | 내용                                                  |
| ---------- | ----------------------------------------------------- |
| `Feat`     | 새로운 기능을 추가할 경우                             |
| `Fix `     | 버그를 고친 경우                                      |
| `Style`    | 코드 포맷 변경, 세미 콜론 누락, 코드 수정이 없는 경우 |
| `Refactor` | 코드 리팩토링                                         |
| `Comment`  | 필요한 주석 추가 및 변경                              |
| `Docs`     | 문서를 수정한 경우                                    |
| `Rename`   | 파일 혹은 폴더명을 수정하거나 옮기는 작업만인 경우    |
| `Remove`   | 파일을 삭제하는 작업만 수행한 경우                    |
| `Setting`  | 개발 환경을 설정, 수정한 경우                         |
